新手小白如何做好一個網站

這篇是專門寫給和我一樣的工程小白看的！我自己就是完全不懂程式，但照著這些步驟，真的做出了一個網站。你也可以的 💪

✓你需要準備什麼

其實沒有你想的那麼多：

🖥️ 一台電腦（Mac 或 Windows 都可以）
🤖 Claude Code（免費，接下來會教你裝）
⏳ 耐心（這個最重要，遇到錯誤不要慌）

就這樣，真的不需要學過任何程式語言！

01安裝環境（只需要做一次）

打開終端機（Terminal），按順序貼上這三段指令，每段按 Enter 等跑完再貼下一段：

① 裝 Homebrew（Mac 的工具管理員）：

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

② 裝 Node.js（Claude Code 需要它才能跑）：

brew install node

③ 裝 Claude Code：

npm install -g @anthropic-ai/claude-code

跑完沒有紅色錯誤訊息就是成功了 🎉 如果空間不足，可以先參考第一篇筆記清快取。

02建立資料夾，啟動 Claude Code

建一個放網站的資料夾，再進去啟動 Claude Code：

mkdir ~/Desktop/my-website
cd ~/Desktop/my-website
claude

看到 Claude Code 的提示符號出現，就代表啟動成功，可以開始和它說話了！

03用說話的方式告訴 Claude 你要什麼

這是我最喜歡的部分——直接打中文就好，把 Claude Code 當成一個超厲害的朋友：

💬「幫我建一個簡單的個人網頁，背景要藍色，標題是我的名字」
💬「幫我在網頁上加一段自我介紹」
💬「把字體改大一點，顏色改成深藍色」

Claude 會直接幫你寫好所有程式碼，你只需要說你想要什麼就好。不用懂任何語法，說人話就行！

04一步一步修改，每次只改一件事

新手最容易犯的錯是一次要改太多，結果不知道哪裡出問題。我的建議是：

✅ 改完一件事就去瀏覽器看效果（雙擊 index.html 就能預覽）
✅ 滿意了再繼續改下一件
✅ 不喜歡就直接說「我不喜歡這個樣子，改回去」

沒有什麼是改不回去的，放心大膽試試看 😄

05上傳到 GitHub Pages，讓全世界都看到

先去 github.com 申請免費帳號。帳號名稱想好，因為你的網址會是 帳號名稱.github.io。

接著安裝 GitHub CLI 並登入：

brew install gh
gh auth login

然後建立 repo 並上傳（把下面的 你的帳號 換成你實際的帳號名稱）：

git init
git add index.html
git commit -m "my first website"
gh repo create 你的帳號.github.io --public
git remote add origin https://github.com/你的帳號/你的帳號.github.io.git
git push -u origin main

幾分鐘後打開 你的帳號.github.io 就能看到你的網站了 🌐

⚠常見錯誤和解決方法

🔴 磁碟空間不足，無法安裝
→ 參考第一篇筆記，用終端機清除 App 快取，通常能釋出好幾 GB 空間。

🔴 指令跑一半卡住不動
→ 等久一點（網路慢很正常），或按 Ctrl + C 取消後重新貼一次。

🔴 看到紅色錯誤訊息
→ 把整段錯誤文字複製，貼給 Claude Code 問「這是什麼意思，怎麼解決？」它會告訴你。

🔴 網站打開是空白的
→ 確認資料夾裡有 index.html，用瀏覽器直接打開確認內容沒問題，再重新 push 一次。

💛給小白的心理建設

說真的，我剛開始也覺得很難，很多指令看不懂，遇到奇怪的錯誤也不知道怎麼辦。但是——

👉 不懂沒關係，把看不懂的東西丟給 Claude Code 問，它不會嫌你問題蠢
👉 錯了可以重來，程式不會爆炸，最多就是重新做一次
👉 每個工程師都是從小白開始的，你只是剛好比他們晚一點開始而已

現在你正在看的這個網站，就是我用 Claude Code 一句一句說出來做的。如果我可以，你也可以 🌱
